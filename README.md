# positionsys
Position system is API for trading bots and gets data

## install 

```
git switch main && git pull
```
and then..
```
pixi install 
```

## testing 
execute all tests (tbd.)
```
pixi run test
```
or 
```
pixi run test::$CASE$
```

## generate documentation

using python sphinx (autodoc) to create a project's documentation

```
pixi run gendoc
```
